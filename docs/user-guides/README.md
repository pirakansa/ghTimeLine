# User Guides

These guides explain how to use `ghStreamListner` in day-to-day workflows.

- [Getting started](getting-started.md): start the app and complete first-run
  setup.
- [Local data](local-data.md): find local configuration and database files.
- [Saved queries](saved-queries.md): create, edit, disable, and delete GitHub
  Search subscriptions.
- [Refreshing streams](refreshing-streams.md): run manual refreshes and adjust
  automatic polling.
- [Stream view](stream-view.md): use library views, filters, sorting, item
  actions, and status messages.
